# Visual Solver Ανισώσεων (Α’ Λυκείου) — Streamlit δραστηριότητα

Μοντέρνα, διαδραστική δραστηριότητα για διδασκαλία ανισώσεων στην Α’ Λυκείου.

**Developed by Nikolaos Sampanis**

---

## ✨ Τι περιλαμβάνει

### 🔍 Εξερεύνηση (Solver + Οπτικοποίηση)
- Γράφεις 1 ή περισσότερες ανισώσεις (μία ανά γραμμή).
- Η εφαρμογή:
  - λύνει στο ℝ (SymPy),
  - δίνει λύση σε **διαστήματα/σύνολα**,
  - εξηγεί **ανοικτά/κλειστά** άκρα,
  - οπτικοποιεί σε **αριθμητική ευθεία**,
  - βρίσκει **κοινή λύση** (τομή) όταν δώσεις πολλές ανισώσεις.

### 🎯 Δραστηριότητα (student-friendly)
- Έτοιμα σενάρια-ασκήσεις (γραμμικές, με απόλυτο, πολυωνυμικές, ρητές).
- Ο μαθητής:
  1) επιλέγει σωστή απάντηση (Multiple Choice),
  2) βλέπει λύση/ευθεία,
  3) διαβάζει σύντομη εξήγηση.

### 🧠 Θεωρία (συνοπτικά)
- Συμβολισμοί διαστημάτων (ανοικτό/κλειστό).
- Απόλυτη τιμή ως «δύο περιπτώσεις».
- Τομή λύσεων (κοινές λύσεις).

```bash
pip install -r requirements.txt
streamlit run app.py
```

## 🧾 Παραδείγματα εισόδου

- `2*x - 3 <= 5`
- `Abs(x-2) < 3`
- `x^2 - 5*x + 6 > 0`
- `(x-1)/(x+2) >= 0`

## ☁️ Deployment (Streamlit Community Cloud)
1. Ανέβασε αυτό το repo σε GitHub.
2. Στο Streamlit Cloud κάνε "New app" και διάλεξε το repo.
3. Στο πεδίο **Main file path** γράψε `app.py`.
